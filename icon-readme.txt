ICON PLACEHOLDER
================

You need to add an icon.png file (128x128px) to this directory.

Quick options:
1. Download a free icon from flaticon.com or icons8.com
2. Create a simple design in any image editor
3. Use this placeholder base64 data URL in manifest.json temporarily:
   data:image/png;base64,iVBORw0KGgoAAAANSUhEUgAAAAEAAAABCAYAAAAfFcSJAAAADUlEQVR42mNk+M9QDwADhgGAWjR9awAAAABJRU5ErkJggg==

Recommended search terms:
- "pen icon"
- "email icon"  
- "quill icon"
- "ghostwriter icon"

The extension will load without an icon but Chrome will show a warning.
